# Contributing Guide

## 1) Directory Rule

- 개인 디렉토리: `members/<your-name>/`
- 주차 파일 예시: `members/<your-name>/week01.md`

## 2) Weekly Content Minimum

- Primary(8th) 페이지 범위 요약
- v9 PPT 대응 슬라이드 핵심 정리
- v8-v9 차이점 3개 이상
- 질문 2개 + 발표 포인트 1개

## 3) PR Rule

- Branch: `feat/<name>-weekXX`
- Title: `[weekXX] <name> - <topic>`
- Body:
  - 범위 (교재 p.xx~yy / PPT chX slide a~b)
  - 이번 주 핵심
  - 확인이 필요한 포인트

## 4) Commit Message (추천)

- `docs(weekXX): add <name> study notes`
