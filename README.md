# Computer Networking 4-1 Study

Kurose & Ross **8th (Primary)** 기반으로 정리하고, 수업에서 사용하는 **9th PPT** 차이를 보강하는 스터디 저장소입니다.

## Repository Structure

- `schedule/` : 주차별 계획표
- `weeks/` : 주차 공통 문서 (week01~week16)
- `members/` : 개인 정리 디렉토리
- `templates/` : 공통 템플릿
- `CONTRIBUTING.md` : PR 규칙

## How We Work

1. 각자 `members/<name>/` 디렉토리를 생성합니다.
2. 매주 `weeks/weekXX.md`를 참고해 개인 정리를 작성합니다.
3. PR 제목 규칙: `[weekXX] 이름 - 핵심주제`
4. 리뷰 후 main에 병합합니다.
